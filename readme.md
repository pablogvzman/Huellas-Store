Huellas Store

Huellas Store es una página web de una tienda de mascotas creada con etiquetas semánticas de HTML5 para lograr una estructura clara, ordenada y accesible.

-Estructura del sitio

Header: contiene el logo y un menú de navegación con enlaces a Inicio, Productos y Contacto.

Main: incluye tres secciones principales:
-Bienvenida: texto introductorio sobre la tienda.
-Productos: tarjetas con imagen, precio, descripción y botón de compra.
-Contacto: formulario para enviar nombre, correo y mensaje.

Footer: dividido en tres partes — mapa de ubicación, información de contacto y enlaces a redes sociales — seguido de una línea con derechos de autor.
